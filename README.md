# gweb

### env prepare
1. install
    1. node.js: https://nodejs.org/zh-cn/download/
    1. cnpm: npm install -g cnpm \[--registry=https://registry.npm.taobao.org\]
    1. vue-cli: cnpm install -g vue-cli
1. init project
    1. vue-init webpack \<front end name\>
    1. cd \<front end name\>
    1. cnpm install
    1. cnpm run dev
    1. cnpm run build
